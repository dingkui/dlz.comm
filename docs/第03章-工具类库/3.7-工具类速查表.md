## 🔍 按功能查找

### 类型转换

```java
// ValUtil - 万能类型转换
ValUtil.toInt(obj);              // 转整数
ValUtil.toStr(obj);              // 转字符串
ValUtil.toList(obj, Class);      // 转列表
ValUtil.toBool(obj);             // 转布尔值
ValUtil.toDouble(obj);           // 转小数
```

### JSON 处理

```java
// JacksonUtil - JSON 序列化
JacksonUtil.toJson(obj);         // 对象 → JSON
JacksonUtil.parse(json, Class);  // JSON → 对象
JacksonUtil.getByPath(json, path); // 路径取值
```

### 日期处理

```java
// DateUtil - 日期时间
DateUtil.now();                  // 当前时间
DateUtil.format(date, pattern);  // 格式化
DateUtil.parse(str, pattern);    // 解析
DateUtil.addDays(date, days);    // 加减天数
```

### 字符串处理

```java
// StringUtils - 字符串操作
StringUtils.isBlank(str);        // 判空
StringUtils.split(str, sep);     // 分割
StringUtils.join(list, sep);     // 连接
StringUtils.trim(str);           // 去空格
```

### Bean 操作

```java
// BeanUtil - Bean 工具
BeanUtil.copyProperties(src, target);     // 属性复制
BeanUtil.copyAsSource(bean, map, false);  // Bean → Map
BeanUtil.copyAsTarget(map, bean);         // Map → Bean
```

### 缓存操作

```java
// Cache - 缓存管理
Cache.get(key);                  // 获取缓存
Cache.put(key, value);           // 设置缓存
Cache.put(key, value, ttl);      // 设置缓存（带过期）
Cache.remove(key);               // 删除缓存
```

## 🎯 常见场景速查

### 场景1：API 参数处理

```java
// 前端传来的参数类型不确定
String ageStr = request.getParameter("age");
Integer age = ValUtil.toInt(ageStr, 0);  // 转整数，失败返回 0

String idsStr = request.getParameter("ids");
List<Integer> ids = ValUtil.toList(idsStr, Integer.class);  // "1,2,3" → [1,2,3]
```

### 场景2：JSON 序列化

```java
// 对象 → JSON
User user = getUser();
String json = JacksonUtil.toJson(user);

// JSON → 对象
String json = getApiResponse();
User user = JacksonUtil.parse(json, User.class);
```

### 场景3：日期格式化

```java
// 当前时间
Date now = DateUtil.now();

// 格式化
String dateStr = DateUtil.format(now, "yyyy-MM-dd HH:mm:ss");

// 解析
Date date = DateUtil.parse("2024-01-01", "yyyy-MM-dd");
```

### 场景4：字符串判空

```java
// 判空（null 或空字符串或空格）
if (StringUtils.isBlank(str)) {
    // 为空
}

// 判非空
if (StringUtils.isNotBlank(str)) {
    // 不为空
}
```

### 场景5：Bean 转换

```java
// VO → Entity
UserVO vo = getFormData();
User user = new User();
BeanUtil.copyProperties(vo, user);

// Entity → VO
User user = userService.getById(id);
UserVO vo = new UserVO();
BeanUtil.copyProperties(user, vo);
```

### 场景6：缓存使用

```java
// 获取缓存，不存在则查询
User user = Cache.get("user:" + id);
if (user == null) {
    user = userService.getById(id);
    Cache.put("user:" + id, user, 3600);  // 缓存 1 小时
}
```

## 📖 API 快速参考

### ValUtil 常用方法

| 方法 | 说明 | 示例 |
|------|------|------|
| `toInt(obj)` | 转整数 | `ValUtil.toInt("123")` → 123 |
| `toInt(obj, def)` | 转整数（带默认值） | `ValUtil.toInt("abc", 0)` → 0 |
| `toLong(obj)` | 转长整数 | `ValUtil.toLong("123")` → 123L |
| `toDouble(obj)` | 转小数 | `ValUtil.toDouble("99.9")` → 99.9 |
| `toStr(obj)` | 转字符串 | `ValUtil.toStr(123)` → "123" |
| `toBool(obj)` | 转布尔值 | `ValUtil.toBool("true")` → true |
| `toList(obj, Class)` | 转列表 | `ValUtil.toList("1,2,3", Integer.class)` |

### JacksonUtil 常用方法

| 方法 | 说明 | 示例 |
|------|------|------|
| `toJson(obj)` | 对象 → JSON | `JacksonUtil.toJson(user)` |
| `parse(json, Class)` | JSON → 对象 | `JacksonUtil.parse(json, User.class)` |
| `parseList(json, Class)` | JSON → 列表 | `JacksonUtil.parseList(json, User.class)` |
| `getByPath(json, path)` | 路径取值 | `JacksonUtil.getByPath(json, "user.name")` |

### DateUtil 常用方法

| 方法 | 说明 | 示例 |
|------|------|------|
| `now()` | 当前时间 | `DateUtil.now()` |
| `format(date, pattern)` | 格式化 | `DateUtil.format(date, "yyyy-MM-dd")` |
| `parse(str, pattern)` | 解析 | `DateUtil.parse("2024-01-01", "yyyy-MM-dd")` |
| `addDays(date, days)` | 加减天数 | `DateUtil.addDays(date, 7)` |
| `addHours(date, hours)` | 加减小时 | `DateUtil.addHours(date, 2)` |

### StringUtils 常用方法

| 方法 | 说明 | 示例 |
|------|------|------|
| `isBlank(str)` | 判空 | `StringUtils.isBlank("")` → true |
| `isNotBlank(str)` | 判非空 | `StringUtils.isNotBlank("abc")` → true |
| `split(str, sep)` | 分割 | `StringUtils.split("a,b,c", ",")` |
| `join(list, sep)` | 连接 | `StringUtils.join(list, ",")` |
| `trim(str)` | 去空格 | `StringUtils.trim(" abc ")` → "abc" |

### BeanUtil 常用方法

| 方法 | 说明 | 示例 |
|------|------|------|
| `copyProperties(src, target)` | 属性复制 | `BeanUtil.copyProperties(vo, user)` |
| `copyAsSource(bean, map, ignoreNull)` | Bean → Map | `BeanUtil.copyAsSource(user, map, false)` |
| `copyAsTarget(map, bean)` | Map → Bean | `BeanUtil.copyAsTarget(map, user)` |

### Cache 常用方法

| 方法 | 说明 | 示例 |
|------|------|------|
| `get(key)` | 获取缓存 | `Cache.get("user:1")` |
| `put(key, value)` | 设置缓存 | `Cache.put("user:1", user)` |
| `put(key, value, ttl)` | 设置缓存（带过期） | `Cache.put("user:1", user, 3600)` |
| `remove(key)` | 删除缓存 | `Cache.remove("user:1")` |
| `clear()` | 清空缓存 | `Cache.clear()` |

## 💡 使用技巧

### 技巧1：链式调用

```java
// ValUtil 支持链式转换
String result = ValUtil.toStr(
    ValUtil.toInt(
        ValUtil.toStr(obj)
    )
);
```

### 技巧2：默认值

```java
// 提供默认值，避免 null
int age = ValUtil.toInt(ageStr, 0);
String name = ValUtil.toStr(nameObj, "未知");
```

### 技巧3：批量转换

```java
// 批量转换列表元素
List<String> strList = Arrays.asList("1", "2", "3");
List<Integer> intList = ValUtil.toList(strList, Integer.class);
```
