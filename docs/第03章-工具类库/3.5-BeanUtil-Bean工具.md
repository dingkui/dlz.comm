# 3.5 BeanUtil - Bean 工具

[← 返回文档导航](../00-文档导航.md)

---

## 📖 简介

`BeanUtil` 提供了 Bean 属性复制、Bean 与 Map 互转等实用功能，配合 `@SetValue` 注解可以实现扁平 Bean 与嵌套 JSON 的双向映射。

---

## 🚀 快速开始

```java
import com.dlz.comm.util.BeanUtil;
import com.dlz.comm.json.JSONMap;

// Bean → Map
User user = new User();
user.setName("张三");
user.setAge(25);

JSONMap map = new JSONMap();
BeanUtil.copyAsSource(user, map, false);
// 结果：{"name":"张三","age":25}

// Map → Bean
JSONMap data = new JSONMap("{\"name\":\"李四\",\"age\":30}");
User user2 = new User();
BeanUtil.copyAsTarget(data, user2);
// user2.name = "李四", user2.age = 30
```

---

## 📚 核心方法

### 1. copyAsSource - Bean 作为数据源

将 Bean 的属性复制到目标对象（Map 或另一个 Bean）。

```java
/**
 * @param source 源 Bean
 * @param target 目标对象（Map 或 Bean）
 * @param ignoreNull 是否忽略 null 值
 */
public static void copyAsSource(Object source, Object target, boolean ignoreNull)
```

**示例：**

```java
User user = new User();
user.setName("张三");
user.setAge(null);

JSONMap map = new JSONMap();

// 不忽略 null
BeanUtil.copyAsSource(user, map, false);
// 结果：{"name":"张三","age":null}

// 忽略 null
BeanUtil.copyAsSource(user, map, true);
// 结果：{"name":"张三"}
```

### 2. copyAsTarget - Bean 作为目标

将源对象（Map 或 Bean）的属性复制到目标 Bean。

```java
/**
 * @param source 源对象（Map 或 Bean）
 * @param target 目标 Bean
 */
public static void copyAsTarget(Object source, Object target)
```

**示例：**

```java
JSONMap data = new JSONMap("{\"name\":\"李四\",\"age\":30}");
User user = new User();

BeanUtil.copyAsTarget(data, user);
// user.name = "李四", user.age = 30
```

### 3. copyProperties - 属性复制

标准的 Bean 属性复制（类似 Spring 的 BeanUtils）。

```java
/**
 * @param source 源 Bean
 * @param target 目标 Bean
 */
public static void copyProperties(Object source, Object target)
```

**示例：**

```java
UserVO vo = new UserVO();
vo.setName("王五");
vo.setEmail("wangwu@example.com");

User user = new User();
BeanUtil.copyProperties(vo, user);
// user.name = "王五", user.email = "wangwu@example.com"
```

## 🎯 配合 @SetValue 注解

`@SetValue` 注解可以指定属性映射到嵌套路径，实现扁平 Bean ↔ 嵌套 JSON 的双向映射。

### 示例：扁平 Bean → 嵌套 JSON

```java
@Data
public class UserForm {
    private String name;
    
    @SetValue("profile")
    private String phone;
    
    @SetValue("profile")
    private String address;
    
    @SetValue("profile.extra")
    private String remark;
}

// 使用
UserForm form = new UserForm();
form.setName("张三");
form.setPhone("13800138000");
form.setAddress("北京");
form.setRemark("VIP用户");

JSONMap json = new JSONMap();
BeanUtil.copyAsSource(form, json, false);

// 结果：
// {
//   "name": "张三",
//   "profile": {
//     "phone": "13800138000",
//     "address": "北京",
//     "extra": {
//       "remark": "VIP用户"
//     }
//   }
// }
```

### 示例：嵌套 JSON → 扁平 Bean

```java
String json = "{" +
    "\"name\":\"李四\"," +
    "\"profile\":{" +
        "\"phone\":\"13900139000\"," +
        "\"address\":\"上海\"," +
        "\"extra\":{\"remark\":\"普通用户\"}" +
    "}" +
"}";

JSONMap data = new JSONMap(json);
UserForm form = new UserForm();
BeanUtil.copyAsTarget(data, form);

// form.name = "李四"
// form.phone = "13900139000"
// form.address = "上海"
// form.remark = "普通用户"
```

详见：[4.1 @SetValue 注解映射](../第04章-高级特性/4.1-SetValue注解映射.md)

## 💡 实战场景

### 场景1：VO 与 Entity 转换

```java
// Entity → VO
User user = userService.getById(id);
UserVO vo = new UserVO();
BeanUtil.copyProperties(user, vo);
return vo;

// VO → Entity
UserVO vo = getFormData();
User user = new User();
BeanUtil.copyProperties(vo, user);
userService.save(user);
```

### 场景2：数据库 JSON 字段处理

```java
@Data
public class Product {
    private Long id;
    private String name;
    
    // 数据库存储为 JSON 字符串
    private String extInfo;
    
    // 扁平属性，映射到 extInfo
    @SetValue("ext")
    private String brand;
    
    @SetValue("ext")
    private String model;
}

// 保存前：Bean → JSON
Product product = new Product();
product.setName("手机");
product.setBrand("Apple");
product.setModel("iPhone 15");

JSONMap ext = new JSONMap();
BeanUtil.copyAsSource(product, ext, true);
product.setExtInfo(ext.toString());
// extInfo = {"ext":{"brand":"Apple","model":"iPhone 15"}}

// 查询后：JSON → Bean
Product product = productMapper.selectById(id);
JSONMap ext = new JSONMap(product.getExtInfo());
BeanUtil.copyAsTarget(ext, product);
// product.brand = "Apple", product.model = "iPhone 15"
```

### 场景3：Excel 导入导出

```java
@Data
public class UserExcel {
    @ExcelProperty("姓名")
    private String name;
    
    @ExcelProperty("手机号")
    @SetValue("profile")
    private String phone;
    
    @ExcelProperty("地址")
    @SetValue("profile")
    private String address;
}

// 导入：Excel → 数据库
List<UserExcel> excelList = EasyExcel.read(file).sheet().doReadSync();
for (UserExcel excel : excelList) {
    JSONMap json = new JSONMap();
    BeanUtil.copyAsSource(excel, json, true);
    
    User user = json.as(User.class);
    userService.save(user);
}

// 导出：数据库 → Excel
List<User> users = userService.list();
List<UserExcel> excelList = new ArrayList<>();
for (User user : users) {
    JSONMap json = new JSONMap(user);
    UserExcel excel = new UserExcel();
    BeanUtil.copyAsTarget(json, excel);
    excelList.add(excel);
}
EasyExcel.write(file, UserExcel.class).sheet().doWrite(excelList);
```

## ⚠️ 注意事项

### 1. 类型转换

BeanUtil 会自动进行类型转换，但复杂类型可能失败：

```java
// 简单类型：自动转换
map.put("age", "25");  // 字符串
BeanUtil.copyAsTarget(map, user);
// user.age = 25（自动转为 Integer）

// 复杂类型：可能失败
map.put("tags", "tag1,tag2");  // 字符串
BeanUtil.copyAsTarget(map, user);
// user.tags 可能为 null（无法自动转为 List）
```

### 2. 属性名匹配

属性名必须完全匹配（区分大小写）：

```java
// ✅ 匹配
source.setUserName("张三");
target.setUserName(...);  // 可以复制

// ❌ 不匹配
source.setUserName("张三");
target.setUsername(...);  // 无法复制（大小写不同）
```

### 3. null 值处理

```java
// ignoreNull = false：复制 null 值
BeanUtil.copyAsSource(source, target, false);

// ignoreNull = true：跳过 null 值
BeanUtil.copyAsSource(source, target, true);
```
