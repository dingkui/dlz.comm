## 📖 集成说明

dlz.comm 与传统 Spring MVC 项目完全兼容。

---

## 🚀 快速示例

```java
@Controller
@RequestMapping("/user")
public class UserController {
    
    @PostMapping("/save")
    @ResponseBody
    public Map<String, Object> save(@RequestBody String formData) {
        JSONMap form = new JSONMap(formData);
        User user = form.as(User.class);
        userService.save(user);
        
        return new JSONMap("code", 200, "message", "success");
    }
}
```
