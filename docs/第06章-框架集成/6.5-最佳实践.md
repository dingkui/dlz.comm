## 📖 使用建议

总结 JSONMap 在实际项目中的最佳实践。

---

## 💡 核心原则

### 1. 优先使用路径表达式

```java
// ✅ 推荐
String name = data.getStr("user.profile.name");

// ❌ 不推荐
JSONMap user = data.getMap("user");
JSONMap profile = user.getMap("profile");
String name = profile.getStr("name");
```

### 2. 提供默认值

```java
// ✅ 推荐
int age = data.getInt("age", 0);

// ❌ 不推荐
Integer age = data.getInt("age");
if (age == null) age = 0;
```

### 3. 使用链式操作

```java
// ✅ 推荐
JSONMap request = new JSONMap()
    .set("header.token", token)
    .set("body.data", data);
```

### 4. 区分 put 和 set

```java
// put：key 不解析
json.put("a.b", 1);  // {"a.b": 1}

// set：key 解析为路径
json.set("a.b", 1);  // {"a": {"b": 1}}
```

---

## ⚠️ 常见错误

### 1. 在循环中创建对象

```java
// ❌ 错误
for (String item : items) {
    JSONMap map = new JSONMap(item);
}

// ✅ 正确
List<JSONMap> maps = items.stream()
    .map(JSONMap::new)
    .collect(Collectors.toList());
```

### 2. 频繁序列化

```java
// ❌ 错误
for (int i = 0; i < 1000; i++) {
    String json = map.toString();
}

// ✅ 正确
String json = map.toString();
```
