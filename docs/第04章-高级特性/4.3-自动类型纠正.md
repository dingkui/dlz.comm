# 4.3 自动类型纠正

[← 返回文档导航](../00-文档导航.md)

---

## 📖 什么是自动类型纠正？

前端传来的数据类型经常不符合预期（如数字传成字符串），JSONMap 可以自动纠正类型，让你的代码更健壮。

---

## 🎯 核心场景

### 场景1：前端类型混乱

```java
// 前端传来的数据
// {"age":"25","price":"99.9","active":"true","count":"3.0"}

JSONMap form = new JSONMap(request.getBody());

// ✨ 自动纠正类型
Integer age = form.getInt("age");           // "25" → 25
Double price = form.getDouble("price");     // "99.9" → 99.9
Boolean active = form.getBoolean("active"); // "true" → true
Integer count = form.getInt("count");       // "3.0" → 3
```

### 场景2：Bean 转换类型纠正

```java
// 前端传来的表单数据
String json = "{\"name\":\"张三\",\"age\":\"25\",\"score\":\"99.5\"}";

JSONMap form = new JSONMap(json);

// 转换为 Bean 时自动纠正类型
User user = form.as(User.class);
// user.age: String "25" → Integer 25
// user.score: String "99.5" → Double 99.5
```

---

## 💡 支持的类型转换

### 字符串 → 数字

```java
JSONMap data = new JSONMap();
data.put("value", "123");

data.getInt("value");        // → 123
data.getLong("value");       // → 123L
data.getDouble("value");     // → 123.0
data.getFloat("value");      // → 123.0f
```

### 字符串 → 布尔值

```java
data.put("flag1", "true");
data.put("flag2", "1");
data.put("flag3", "yes");

data.getBoolean("flag1");    // → true
data.getBoolean("flag2");    // → true
data.getBoolean("flag3");    // → true
```

### 数字 → 字符串

```java
data.put("count", 100);
data.put("price", 99.9);

data.getStr("count");        // → "100"
data.getStr("price");        // → "99.9"
```

### 逗号分隔 → 数组

```java
data.put("ids", "1,2,3,4,5");
data.put("tags", "java,python,go");

List<Integer> ids = data.getList("ids", Integer.class);
// → [1, 2, 3, 4, 5]

List<String> tags = data.getList("tags", String.class);
// → ["java", "python", "go"]
```

## 🚀 实战示例

### 示例1：表单提交

```java
@PostMapping("/user/save")
public R save(@RequestBody String formData) {
    JSONMap form = new JSONMap(formData);
    
    // 即使前端传的都是字符串，也能正确转换
    String name = form.getStr("name");
    Integer age = form.getInt("age");           // "25" → 25
    Boolean active = form.getBoolean("active"); // "true" → true
    Double salary = form.getDouble("salary");   // "8000.5" → 8000.5
    
    User user = new User();
    user.setName(name);
    user.setAge(age);
    user.setActive(active);
    user.setSalary(salary);
    
    userService.save(user);
    return R.success();
}
```

### 示例2：API 参数处理

```java
@GetMapping("/products")
public R list(
    @RequestParam String categoryId,  // 前端可能传 "123" 或 123
    @RequestParam String minPrice,    // 前端可能传 "99.9" 或 99.9
    @RequestParam String maxPrice
) {
    JSONMap params = new JSONMap(
        "categoryId", categoryId,
        "minPrice", minPrice,
        "maxPrice", maxPrice
    );
    
    // 自动纠正类型
    Long catId = params.getLong("categoryId");
    Double min = params.getDouble("minPrice");
    Double max = params.getDouble("maxPrice");
    
    List<Product> products = productService.list(catId, min, max);
    return R.data(products);
}
```

### 示例3：配置文件读取

```java
// 配置文件中的值都是字符串
Properties props = new Properties();
props.load(inputStream);

JSONMap config = new JSONMap(props);

// 自动转换为正确类型
Integer port = config.getInt("server.port");           // "8080" → 8080
Boolean debug = config.getBoolean("app.debug");        // "true" → true
List<String> hosts = config.getList("db.hosts", String.class);  // "host1,host2" → ["host1","host2"]
```

## ⚠️ 注意事项

### 1. 转换失败返回 null

```java
JSONMap data = new JSONMap();
data.put("value", "abc");  // 无法转换为数字

Integer num = data.getInt("value");  // → null（转换失败）
```

### 2. 使用默认值

```java
// 推荐：提供默认值
Integer age = data.getInt("age", 0);

// 不推荐：手动判空
Integer age = data.getInt("age");
if (age == null) {
    age = 0;
}
```

### 3. 精度问题

```java
data.put("value", "99.999999999");

// 可能有精度损失
Double d = data.getDouble("value");

// 精确计算用 BigDecimal
BigDecimal bd = data.getBigDecimal("value");
```

## 💡 最佳实践

### 1. 前端接口统一处理

```java
@RestControllerAdvice
public class RequestBodyAdvice implements RequestBodyAdvice {
    
    @Override
    public Object afterBodyRead(Object body, ...) {
        // 统一转换为 JSONMap，自动纠正类型
        if (body instanceof String) {
            return new JSONMap((String) body);
        }
        return body;
    }
}
```

### 2. 参数验证

```java
JSONMap form = new JSONMap(request.getBody());

// 先验证，再转换
if (form.getInt("age") == null) {
    throw new BusinessException("年龄格式错误");
}

Integer age = form.getInt("age");
```

### 3. 日志记录

```java
JSONMap form = new JSONMap(request.getBody());

// 记录原始数据
log.info("原始数据：{}", form.toString());

// 记录转换后的数据
Integer age = form.getInt("age");
log.info("转换后年龄：{}", age);
```
