# 4.2 多维数组支持

[← 返回文档导航](../00-文档导航.md)

---

## 📖 简介

JSONMap 支持多维数组的构建和访问，可以轻松处理矩阵、表格等复杂数据结构。

---

## 🚀 快速开始

### 二维数组

```java
JSONMap json = new JSONMap();

// 构建二维数组
json.set("matrix[0][0]", 1);
json.set("matrix[0][1]", 2);
json.set("matrix[1][0]", 3);
json.set("matrix[1][1]", 4);

// 结果：{"matrix":[[1,2],[3,4]]}

// 访问
json.getInt("matrix[0][1]");  // → 2
json.getInt("matrix[1][0]");  // → 3
```

### 三维数组

```java
JSONMap json = new JSONMap();

// 构建三维数组
json.set("cube[0][0][0]", 1);
json.set("cube[0][0][1]", 2);
json.set("cube[0][1][0]", 3);
json.set("cube[1][0][0]", 4);

// 结果：{"cube":[[[1,2],[3]],[[4]]]}

// 访问
json.getInt("cube[0][0][1]");  // → 2
```

## 💡 实战场景

### 场景1：矩阵数据

```java
// 构建 3x3 矩阵
JSONMap matrix = new JSONMap();
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        matrix.set("data[" + i + "][" + j + "]", i * 3 + j + 1);
    }
}

// 结果：
// {
//   "data": [
//     [1, 2, 3],
//     [4, 5, 6],
//     [7, 8, 9]
//   ]
// }

// 访问
int value = matrix.getInt("data[1][1]");  // → 5
```

### 场景2：表格数据

```java
// Excel 表格数据
JSONMap table = new JSONMap();

// 表头
table.set("rows[0][0]", "姓名");
table.set("rows[0][1]", "年龄");
table.set("rows[0][2]", "城市");

// 数据行
table.set("rows[1][0]", "张三");
table.set("rows[1][1]", 25);
table.set("rows[1][2]", "北京");

table.set("rows[2][0]", "李四");
table.set("rows[2][1]", 30);
table.set("rows[2][2]", "上海");

// 访问
String name = table.getStr("rows[1][0]");  // → "张三"
Integer age = table.getInt("rows[1][1]");  // → 25
```

### 场景3：多维配置

```java
// 服务器集群配置
JSONMap config = new JSONMap();

// 第一个集群的第一个服务器的第一个端口
config.set("clusters[0].servers[0].ports[0]", 8080);
config.set("clusters[0].servers[0].ports[1]", 8081);

// 第一个集群的第二个服务器
config.set("clusters[0].servers[1].ports[0]", 9080);

// 第二个集群
config.set("clusters[1].servers[0].ports[0]", 7080);

// 访问
Integer port = config.getInt("clusters[0].servers[0].ports[1]");  // → 8081
```

## 🎯 数组自动补齐

当设置的索引超过当前数组长度时，会自动补齐中间元素（填充 null）：

```java
JSONMap json = new JSONMap();

// 直接设置索引 5
json.set("arr[5]", "value5");

// 结果：{"arr":[null,null,null,null,null,"value5"]}

// 多维数组也支持
json.set("matrix[3][3]", 9);
// 结果：{"matrix":[null,null,null,[null,null,null,9]]}
```

---

## ⚠️ 注意事项

### 1. 性能考虑

多维数组的构建和访问会有一定性能开销，建议：

```java
// ✅ 推荐：批量构建
JSONMap matrix = new JSONMap();
for (int i = 0; i < rows; i++) {
    for (int j = 0; j < cols; j++) {
        matrix.set("data[" + i + "][" + j + "]", getValue(i, j));
    }
}

// ❌ 不推荐：频繁创建新对象
for (int i = 0; i < rows; i++) {
    for (int j = 0; j < cols; j++) {
        new JSONMap().set("data[" + i + "][" + j + "]", getValue(i, j));
    }
}
```

### 2. 索引越界

访问不存在的索引返回 null，不抛异常：

```java
JSONMap json = new JSONMap("{\"arr\":[[1,2],[3,4]]}");

json.getInt("arr[0][5]");  // → null（列越界）
json.getInt("arr[5][0]");  // → null（行越界）
```

### 3. 类型混合

同一数组中可以包含不同类型的元素：

```java
JSONMap json = new JSONMap();
json.set("arr[0]", 1);
json.set("arr[1]", "text");
json.set("arr[2]", true);

// 结果：{"arr":[1,"text",true]}
```
