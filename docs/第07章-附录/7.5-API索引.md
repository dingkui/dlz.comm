## 📚 JSONMap API

### 构造方法
- `JSONMap()` - 空对象
- `JSONMap(Object obj)` - 从对象构造
- `JSONMap(String json)` - 从 JSON 字符串构造
- `JSONMap(String key, Object val, ...)` - 键值对构造

### 读取方法
- `getStr(String key)` - 获取字符串
- `getInt(String key)` - 获取整数
- `getLong(String key)` - 获取长整数
- `getDouble(String key)` - 获取小数
- `getBoolean(String key)` - 获取布尔值
- `getMap(String key)` - 获取子对象
- `getList(String key)` - 获取数组
- `as(Class<T> clazz)` - 转换为 Bean

### 写入方法
- `put(String key, Object value)` - 普通写入
- `set(String key, Object value)` - 深层写入
- `add(String key, Object obj)` - 数组追加

---

## 📚 JSONList API

### 构造方法
- `JSONList()` - 空列表
- `JSONList(String json)` - 从 JSON 字符串构造
- `JSONList(Collection<?> collection)` - 从集合构造

### 读取方法
- `getStr(int index)` - 获取字符串
- `getInt(int index)` - 获取整数
- `getMap(int index)` - 获取子对象
- `asList(Class<T> clazz)` - 转换为列表

---

## 📚 ValUtil API

- `toInt(Object obj)` - 转整数
- `toStr(Object obj)` - 转字符串
- `toList(Object obj, Class<T> clazz)` - 转列表
- `toBool(Object obj)` - 转布尔值

---

## 📚 JacksonUtil API

- `toJson(Object obj)` - 对象 → JSON
- `parse(String json, Class<T> clazz)` - JSON → 对象
- `getByPath(String json, String path)` - 路径取值

---

## 📚 DateUtil API

- `now()` - 当前时间
- `format(Date date, String pattern)` - 格式化
- `parse(String str, String pattern)` - 解析
- `addDays(Date date, int days)` - 加减天数

---

## 📚 StringUtils API

- `isBlank(String str)` - 判空
- `split(String str, String sep)` - 分割
- `join(List<?> list, String sep)` - 连接

---

## 📚 BeanUtil API

- `copyProperties(Object src, Object target)` - 属性复制
- `copyAsSource(Object bean, Object map, boolean ignoreNull)` - Bean → Map
- `copyAsTarget(Object map, Object bean)` - Map → Bean
