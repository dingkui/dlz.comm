## 🤝 如何贡献

欢迎贡献代码、提出建议或报告问题！

---

## 📝 贡献方式

### 1. 提交 Issue

发现 Bug 或有功能建议？[提交 Issue](https://github.com/dingkui/dlz.comm/issues)

**Issue 模板：**
```
**问题描述**
简要描述问题

**复现步骤**
1. 步骤1
2. 步骤2

**期望行为**
期望的结果

**实际行为**
实际的结果

**环境信息**
- JDK 版本：
- dlz.comm 版本：
- 操作系统：
```

### 2. 提交 Pull Request

1. Fork 本仓库
2. 创建特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 提交 Pull Request

---

## 📖 代码规范

### 命名规范

- 类名：大驼峰（PascalCase）
- 方法名：小驼峰（camelCase）
- 常量：全大写下划线（UPPER_SNAKE_CASE）

### 注释规范

```java
/**
 * 方法说明
 * 
 * @param key 参数说明
 * @return 返回值说明
 */
public String getStr(String key) {
    // 实现
}
```
