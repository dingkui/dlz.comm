# 2.3 深层路径详解

[← 返回文档导航](../00-文档导航.md)

---

## 📖 什么是深层路径？

深层路径是 JSONMap 的核心特性，允许你用一个字符串表达式直接访问嵌套数据，无需层层判空和强转。

```java
// 传统方式：20 行代码
String city = null;
if (data != null && data.get("user") != null) {
    Map user = (Map) data.get("user");
    if (user.get("profile") != null) {
        Map profile = (Map) user.get("profile");
        if (profile.get("addresses") != null) {
            List addresses = (List) profile.get("addresses");
            if (!addresses.isEmpty()) {
                Map addr = (Map) addresses.get(0);
                city = (String) addr.get("city");
            }
        }
    }
}

// JSONMap 方式：1 行代码
String city = data.getStr("user.profile.addresses[0].city");
```

## 🎯 路径语法规则

### 基本语法

| 语法 | 说明 | 示例 |
|------|------|------|
| `.` | 访问对象属性 | `user.name` |
| `[n]` | 访问数组索引 n | `users[0]` |
| `[-n]` | 访问倒数第 n 个 | `users[-1]` |
| 混合 | 任意组合 | `users[0].profile.tags[-1]` |

### 详细说明

#### 1. 点号访问（`.`）

用于访问对象的属性：

```java
JSONMap data = new JSONMap("{\"user\":{\"name\":\"张三\",\"age\":25}}");

data.getStr("user");          // → {"name":"张三","age":25}
data.getStr("user.name");     // → "张三"
data.getInt("user.age");      // → 25
```

#### 2. 数组索引（`[n]`）

用于访问数组的第 n 个元素（从 0 开始）：

```java
JSONMap data = new JSONMap("{\"tags\":[\"a\",\"b\",\"c\"]}");

data.getStr("tags[0]");       // → "a"
data.getStr("tags[1]");       // → "b"
data.getStr("tags[2]");       // → "c"
```

#### 3. 负数索引（`[-n]`）

用于访问倒数第 n 个元素：

```java
JSONMap data = new JSONMap("{\"tags\":[\"a\",\"b\",\"c\",\"d\",\"e\"]}");

data.getStr("tags[-1]");      // → "e"（最后一个）
data.getStr("tags[-2]");      // → "d"（倒数第二个）
data.getStr("tags[-5]");      // → "a"（倒数第五个）
```

#### 4. 混合路径

可以任意组合点号和下标：

```java
String json = "{" +
    "\"users\":[" +
        "{\"name\":\"张三\",\"profile\":{\"city\":\"北京\",\"tags\":[\"vip\",\"active\"]}}," +
        "{\"name\":\"李四\",\"profile\":{\"city\":\"上海\",\"tags\":[\"normal\"]}}" +
    "]" +
"}";

JSONMap data = new JSONMap(json);

// 复杂路径
data.getStr("users[0].name");                    // → "张三"
data.getStr("users[0].profile.city");            // → "北京"
data.getStr("users[0].profile.tags[0]");         // → "vip"
data.getStr("users[-1].profile.tags[-1]");       // → "normal"
```

## 💡 高级用法

### 1. 多维数组

```java
JSONMap json = new JSONMap();

// 构建二维数组
json.set("matrix[0][0]", 1);
json.set("matrix[0][1]", 2);
json.set("matrix[1][0]", 3);
json.set("matrix[1][1]", 4);

// 结果：{"matrix":[[1,2],[3,4]]}

// 访问
json.getInt("matrix[0][1]");  // → 2
json.getInt("matrix[1][0]");  // → 3
```

### 2. 数组中的对象数组

```java
String json = "{" +
    "\"departments\":[" +
        "{\"name\":\"技术部\",\"members\":[{\"name\":\"张三\"},{\"name\":\"李四\"}]}," +
        "{\"name\":\"市场部\",\"members\":[{\"name\":\"王五\"}]}" +
    "]" +
"}";

JSONMap data = new JSONMap(json);

// 访问第一个部门的第二个成员
String name = data.getStr("departments[0].members[1].name");  // → "李四"

// 访问最后一个部门的最后一个成员
String lastName = data.getStr("departments[-1].members[-1].name");  // → "王五"
```

### 3. 路径中的特殊字符

如果键名包含点号或方括号，需要特殊处理：

```java
// 键名包含点号
JSONMap data = new JSONMap();
data.put("user.name", "张三");  // 键名就是 "user.name"

// 不能用路径访问
data.getStr("user.name");  // ❌ 会被解析为 user → name

// 应该直接用 get
data.get("user.name");     // ✅ 正确
```

## ⚠️ 注意事项

### 1. 路径不存在

路径中任意一环不存在，返回 null，不抛异常：

```java
JSONMap data = new JSONMap("{\"user\":{\"name\":\"张三\"}}");

data.getStr("user.age");           // → null（age 不存在）
data.getStr("user.profile.city");  // → null（profile 不存在）
data.getStr("admin.name");         // → null（admin 不存在）
```

### 2. 类型不匹配

如果路径指向的不是预期类型，返回 null：

```java
JSONMap data = new JSONMap("{\"user\":{\"name\":\"张三\"}}");

// user.name 是字符串，不是对象
data.getMap("user.name");          // → null
data.getList("user.name");         // → null
```

### 3. 数组越界

数组索引越界时，返回 null：

```java
JSONMap data = new JSONMap("{\"tags\":[\"a\",\"b\",\"c\"]}");

data.getStr("tags[10]");           // → null（越界）
data.getStr("tags[-10]");          // → null（越界）
```

### 4. put vs set

```java
JSONMap json = new JSONMap();

// put：key 不解析，直接作为键名
json.put("a.b", 1);
// 结果：{"a.b": 1}

// set：key 解析为路径
json.set("a.b", 1);
// 结果：{"a": {"b": 1}}
```

## 🎯 最佳实践

### 1. 优先使用路径表达式

```java
// ✅ 推荐
String name = data.getStr("user.profile.name");

// ❌ 不推荐
JSONMap user = data.getMap("user");
JSONMap profile = user.getMap("profile");
String name = profile.getStr("name");
```

### 2. 提供默认值

```java
// ✅ 推荐
String city = data.getStr("user.profile.city", "未知");

// ❌ 不推荐
String city = data.getStr("user.profile.city");
if (city == null) {
    city = "未知";
}
```

### 3. 使用负索引获取最后元素

```java
// ✅ 推荐
String lastTag = data.getStr("tags[-1]");

// ❌ 不推荐
JSONList tags = data.getList("tags");
String lastTag = tags.getStr(tags.size() - 1);
```
