# 2.2 JSONList 完整指南

[← 返回文档导航](../00-文档导航.md)

---

## 📖 简介

`JSONList` 是增强版的 `ArrayList`，专门为 JSON 数组处理而设计。它提供了：

- 🎯 **负索引访问**：Python 风格的数组访问
- 🔄 **类型安全访问**：自动类型转换
- 🔗 **链式操作**：流畅的 API 设计

---

## 🚀 快速开始

```java
import com.dlz.comm.json.JSONList;

// 创建 JSONList
JSONList list = new JSONList("[1, 2, 3, 4, 5]");

// 正向索引
list.getInt(0);    // → 1（第一个）
list.getInt(1);    // → 2（第二个）

// 负向索引（Python 风格）
list.getInt(-1);   // → 5（最后一个）
list.getInt(-2);   // → 4（倒数第二个）
```

---

## 📝 构造方式

```java
// 1️⃣ 空列表
JSONList list = new JSONList();

// 2️⃣ JSON 字符串
JSONList list = new JSONList("[1, 2, 3]");

// 3️⃣ 逗号分隔字符串（自动拆分！）
JSONList list = new JSONList("苹果,香蕉,橙子");
// 结果：["苹果", "香蕉", "橙子"]

// 4️⃣ 数组
JSONList list = new JSONList(new String[]{"a", "b", "c"});

// 5️⃣ 集合
JSONList list = new JSONList(Arrays.asList(1, 2, 3));

// 6️⃣ 带类型转换
JSONList list = new JSONList("1,2,3", Integer.class);
// 结果：[1, 2, 3]（整数类型）

// 7️⃣ 指定初始容量
JSONList list = new JSONList(100);
```

## 🎯 核心特性

### 1. 负索引访问

```java
JSONList list = new JSONList("[10, 20, 30, 40, 50]");

// 正向索引
list.getInt(0);     // → 10（第一个）
list.getInt(1);     // → 20（第二个）
list.getInt(4);     // → 50（第五个）

// 负向索引
list.getInt(-1);    // → 50（最后一个）
list.getInt(-2);    // → 40（倒数第二个）
list.getInt(-5);    // → 10（倒数第五个）
```

### 2. 类型安全访问

```java
JSONList list = new JSONList("[{\"name\":\"张三\",\"age\":25}, {\"name\":\"李四\",\"age\":30}]");

// 获取子对象
JSONMap first = list.getMap(0);
String name = first.getStr("name");  // → "张三"

// 直接用路径
String lastName = list.getStr("[-1].name");  // → "李四"（最后一个人的名字）
```

### 3. 链式操作

```java
JSONList list = new JSONList()
    .adds("元素1")
    .adds("元素2")
    .adds("元素3");
```

## 💡 实战示例

### 示例1：遍历数组

```java
JSONList users = new JSONList("[" +
    "{\"name\":\"张三\",\"age\":25}," +
    "{\"name\":\"李四\",\"age\":30}," +
    "{\"name\":\"王五\",\"age\":28}" +
    "]");

// 方式1：使用索引
for (int i = 0; i < users.size(); i++) {
    String name = users.getStr(i + ".name");
    Integer age = users.getInt(i + ".age");
    System.out.println(name + " - " + age + "岁");
}

// 方式2：转换为 Bean 列表
List<User> userList = users.asList(User.class);
for (User user : userList) {
    System.out.println(user.getName() + " - " + user.getAge() + "岁");
}
```

### 示例2：获取最后几个元素

```java
JSONList logs = new JSONList(logArray);

// 获取最后一条日志
JSONMap lastLog = logs.getMap(-1);

// 获取最后3条日志
List<JSONMap> last3 = new ArrayList<>();
for (int i = -3; i < 0; i++) {
    last3.add(logs.getMap(i));
}
```

### 示例3：逗号分隔字符串处理

## ⚠️ 注意事项

### 1. 负索引范围

```java
JSONList list = new JSONList("[1, 2, 3]");

list.getInt(-1);   // ✅ 正确：3
list.getInt(-3);   // ✅ 正确：1
list.getInt(-4);   // ❌ 越界：IndexOutOfBoundsException
```

### 2. 类型转换失败

```java
JSONList list = new JSONList("[\"abc\", \"def\"]");

Integer num = list.getInt(0);  // → null（转换失败返回 null）
```

### 3. 空值处理

```java
JSONList list = new JSONList("[1, null, 3]");

Integer val = list.getInt(1);  // → null
Integer val = list.getInt(1, 0);  // → 0（使用默认值）
```
