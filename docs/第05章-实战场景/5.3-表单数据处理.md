## 📖 场景说明

处理前端表单提交时，JSONMap 可以自动纠正类型、验证数据、转换为 Bean。

---

## 🚀 快速示例

```java
@PostMapping("/user/save")
public R save(@RequestBody String formData) {
    JSONMap form = new JSONMap(formData);
    
    // 自动类型转换
    String name = form.getStr("name");
    Integer age = form.getInt("age");  // "25" → 25
    Boolean active = form.getBoolean("active");  // "true" → true
    
    // 转换为 Bean
    User user = form.as(User.class);
    userService.save(user);
    
    return R.success();
}
```

## 💡 实战案例

### 案例1：用户注册

```java
@PostMapping("/register")
public R register(@RequestBody String formData) {
    JSONMap form = new JSONMap(formData);
    
    // 验证必填项
    if (form.getStr("username") == null) {
        return R.fail("用户名不能为空");
    }
    
    // 验证格式
    String email = form.getStr("email");
    if (!EmailValidator.isValid(email)) {
        return R.fail("邮箱格式错误");
    }
    
    // 保存用户
    User user = form.as(User.class);
    userService.register(user);
    
    return R.success();
}
```
