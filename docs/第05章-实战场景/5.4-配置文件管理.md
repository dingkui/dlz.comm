# 5.4 配置文件管理

[← 返回文档导航](../00-文档导航.md)

---

## 📖 场景说明

使用 JSONMap 处理配置文件，支持嵌套配置、动态配置、配置合并等。

---

## 🚀 快速示例

```java
// 读取配置文件
String configJson = FileUtils.readFileToString("config.json");
JSONMap config = new JSONMap(configJson);

// 读取配置
int port = config.getInt("server.port", 8080);
String host = config.getStr("server.host", "localhost");
String dbUrl = config.getStr("database.master.url");
List<String> whitelist = config.getList("security.whitelist", String.class);
```

---

## 💡 实战案例

### 案例1：多环境配置

```java
// 加载基础配置
JSONMap baseConfig = new JSONMap(readFile("config-base.json"));

// 加载环境配置
String env = System.getProperty("env", "dev");
JSONMap envConfig = new JSONMap(readFile("config-" + env + ".json"));

// 合并配置（环境配置覆盖基础配置）
baseConfig.putAll(envConfig);

// 使用配置
int port = baseConfig.getInt("server.port");
```

### 案例2：动态配置

```java
// 从配置中心获取配置
String remoteConfig = configCenter.getConfig("app");
JSONMap config = new JSONMap(remoteConfig);

// 监听配置变化
configCenter.addListener(newConfig -> {
    config.clear();
    config.putAll(new JSONMap(newConfig));
    reloadApp(config);
});
```
