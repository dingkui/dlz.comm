## 📝 第一个示例

### 示例1：深层路径访问

```java
import com.dlz.comm.json.JSONMap;

public class QuickStart {
    public static void main(String[] args) {
        // 创建一个 JSONMap
        String json = "{\"user\":{\"name\":\"张三\",\"profile\":{\"city\":\"北京\",\"age\":25}}}";
        JSONMap data = new JSONMap(json);
        
        // ✨ 一行代码穿透多层嵌套
        String name = data.getStr("user.name");              // → "张三"
        String city = data.getStr("user.profile.city");      // → "北京"
        Integer age = data.getInt("user.profile.age");       // → 25
        
        System.out.println("姓名：" + name);
        System.out.println("城市：" + city);
        System.out.println("年龄：" + age);
    }
}
```

**输出：**
```
姓名：张三
城市：北京
年龄：25
```

**对比传统方式：**

```java
// ❌ 传统方式：需要 10+ 行代码
Map<String, Object> data = parseJson(json);
String city = null;
if (data != null && data.get("user") != null) {
    Map user = (Map) data.get("user");
    if (user.get("profile") != null) {
        Map profile = (Map) user.get("profile");
        city = (String) profile.get("city");
    }
}

// ✅ JSONMap 方式：1 行代码
String city = new JSONMap(json).getStr("user.profile.city");
```

### 示例2：智能构建复杂结构

```java
import com.dlz.comm.json.JSONMap;

public class BuildExample {
    public static void main(String[] args) {
        // ✨ 自动创建所有中间层级
        JSONMap config = new JSONMap()
            .set("server.host", "localhost")
            .set("server.port", 8080)
            .set("db.master.url", "jdbc:mysql://localhost:3306/test")
            .set("db.master.username", "root")
            .add("whitelist", "192.168.1.1")
            .add("whitelist", "192.168.1.2");
        
        // 输出 JSON
        System.out.println(config.toString());
    }
}
```

**输出：**
```json
{
  "server": {
    "host": "localhost",
    "port": 8080
  },
  "db": {
    "master": {
      "url": "jdbc:mysql://localhost:3306/test",
      "username": "root"
    }
  },
  "whitelist": ["192.168.1.1", "192.168.1.2"]
}
```

### 示例3：类型自动转换

```java
import com.dlz.comm.json.JSONMap;

public class TypeConversionExample {
    public static void main(String[] args) {
        JSONMap data = new JSONMap();
        data.put("score", "99.5");      // 存的是字符串
        data.put("count", 100);         // 存的是整数
        
        // ✨ 随心所欲地取，自动转换类型
        Integer scoreInt = data.getInt("score");           // → 99
        Double scoreDouble = data.getDouble("score");      // → 99.5
        String countStr = data.getStr("count");            // → "100"
        Long countLong = data.getLong("count");            // → 100L
        
        System.out.println("scoreInt: " + scoreInt);
        System.out.println("scoreDouble: " + scoreDouble);
        System.out.println("countStr: " + countStr);
        System.out.println("countLong: " + countLong);
    }
}
```

**输出：**
```
scoreInt: 99
scoreDouble: 99.5
countStr: 100
countLong: 100
```

### 示例4：处理数组

```java
import com.dlz.comm.json.JSONMap;
import com.dlz.comm.json.JSONList;

public class ArrayExample {
    public static void main(String[] args) {
        String json = "{\"users\":[" +
            "{\"name\":\"张三\",\"age\":25}," +
            "{\"name\":\"李四\",\"age\":30}," +
            "{\"name\":\"王五\",\"age\":28}" +
            "]}";
        
        JSONMap data = new JSONMap(json);
        
        // ✨ 访问数组元素
        String firstName = data.getStr("users[0].name");    // → "张三"
        String lastName = data.getStr("users[-1].name");    // → "王五"（负索引！）
        
        // 获取数组
        JSONList users = data.getList("users");
        System.out.println("用户数量：" + users.size());
        
        // 遍历数组
        for (int i = 0; i < users.size(); i++) {
            String name = users.getStr(i + ".name");
            Integer age = users.getInt(i + ".age");
            System.out.println(name + " - " + age + "岁");
        }
    }
}
```

**输出：**
```
用户数量：3
张三 - 25岁
李四 - 30岁
王五 - 28岁
```

## 🎓 核心概念速览

### 1. 深层路径访问

使用点号 `.` 和下标 `[]` 访问嵌套数据：

```java
data.getStr("user.profile.city");           // 点号访问对象
data.getStr("users[0].name");               // 下标访问数组
data.getStr("users[-1].name");              // 负索引（倒数第一个）
data.getStr("config.servers[0].ports[1]");  // 混合路径
```

### 2. 智能构建

使用 `set()` 自动创建中间层级：

```java
map.set("a.b.c", "value");                  // 自动创建 a → b → c
map.set("arr[0].name", "张三");             // 自动创建数组和对象
map.add("tags", "tag1");                    // 自动创建数组并追加
```

### 3. 类型转换

自动转换类型，无需手动强转：

```java
map.getStr(key);        // 转字符串
map.getInt(key);        // 转整数
map.getDouble(key);     // 转小数
map.getBoolean(key);    // 转布尔值
map.getList(key);       // 转数组
map.getMap(key);        // 转对象
```

## 💡 实用技巧

### 技巧1：提供默认值

```java
String name = data.getStr("user.name", "未知");
Integer age = data.getInt("user.age", 0);
Boolean active = data.getBoolean("user.active", false);
```

### 技巧2：链式操作

```java
JSONMap request = new JSONMap()
    .set("header.token", token)
    .set("header.timestamp", System.currentTimeMillis())
    .set("body.userId", userId)
    .set("body.action", "query");
```

### 技巧3：Bean 转换

```java
// Bean → JSONMap
User user = new User();
JSONMap map = new JSONMap(user);

// JSONMap → Bean
User user = map.as(User.class);
```

### 技巧4：简化构造

```java
// 键值对构造（最简洁）
JSONMap map = new JSONMap(
    "name", "张三",
    "age", 25,
    "city", "北京"
);
```
