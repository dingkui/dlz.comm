## 📦 Maven 依赖（推荐）

在项目的 `pom.xml` 文件中添加以下依赖：

```xml
<dependency>
    <groupId>top.dlzio</groupId>
    <artifactId>dlz.comm</artifactId>
    <version>6.5.1</version>
</dependency>
```

### 最新版本

查看最新版本：[Maven Central](https://search.maven.org/artifact/top.dlzio/dlz.comm)

---

## 🎯 Gradle 依赖

在项目的 `build.gradle` 文件中添加：

```gradle
dependencies {
    implementation 'top.dlzio:dlz.comm:6.5.1'
}
```

或者使用 Kotlin DSL (`build.gradle.kts`)：

```kotlin
dependencies {
    implementation("top.dlzio:dlz.comm:6.5.1")
}
```

---

## 📥 手动下载

如果不使用构建工具，可以手动下载 JAR 文件：

1. 访问 [Maven Central Repository](https://repo1.maven.org/maven2/top/dlzio/dlz.comm/)
2. 选择版本目录（如 `6.5.1/`）
3. 下载 `dlz.comm-6.5.1.jar`
4. 将 JAR 文件添加到项目的 classpath

⚠️ **注意**：手动下载时需要同时下载依赖的 `jackson-databind` 库。

---

## 🔧 依赖说明

dlz.comm 只有一个外部依赖：

```xml
<dependency>
    <groupId>com.fasterxml.jackson.core</groupId>
    <artifactId>jackson-databind</artifactId>
    <version>2.13.0+</version>
</dependency>
```

### 依赖冲突处理

大多数 Spring Boot 项目默认已包含 Jackson，无需额外配置。

如果遇到版本冲突，可以在 Maven 中排除：

```xml
<dependency>
    <groupId>top.dlzio</groupId>
    <artifactId>dlz.comm</artifactId>
    <version>6.5.1</version>
    <exclusions>
        <exclusion>
            <groupId>com.fasterxml.jackson.core</groupId>
            <artifactId>jackson-databind</artifactId>
        </exclusion>
    </exclusions>
</dependency>
```

## ✅ 验证安装

创建一个简单的测试类验证安装是否成功：

```java
import com.dlz.comm.json.JSONMap;

public class InstallTest {
    public static void main(String[] args) {
        JSONMap map = new JSONMap("name", "张三", "age", 25);
        System.out.println(map.getStr("name"));  // 输出：张三
        System.out.println(map.getInt("age"));   // 输出：25
        System.out.println("✅ dlz.comm 安装成功！");
    }
}
```

如果能正常运行并输出结果，说明安装成功。

## 🔍 常见问题

### Q: 我的项目没有 Jackson 怎么办？

A: 添加 dlz.comm 会自动引入 Jackson 依赖，无需额外配置。

### Q: 与 Spring Boot 版本兼容吗？

A: 完全兼容 Spring Boot 2.x 和 3.x。dlz.comm 不依赖 Spring，只依赖 Jackson。

### Q: 支持哪些 JDK 版本？

A: 支持 JDK 8、11、17、21 等所有主流版本。

### Q: 包大小是多少？

A: 约 100KB，非常轻量，不会给项目增加负担。

### Q: 是否需要配置文件？

A: 不需要。dlz.comm 开箱即用，无需任何配置。
